Задача 9

У вас есть исходный код приложения, которое читает данные соискателей из файла, и выполняет поиск по должности среди соискателей. Пользователь указывает должность, которую он ищет с помощью GET-формы.

Вам требуется сделать аналогичное приложение, но, чтобы форма и представление обрабатывали POST-запрос.

Каждая запись о соискателе содержит следующие поля:

  • name - ФИО
  • picture - фото кандидата
  • position - должность
  • skills - список навыков
  • age - возраст
